Codex Usage Overlay for Windows

1. Extract this entire folder.
2. Double-click run_overlay.bat.
3. The tray icon will appear in the Windows notification area. If it is not visible,
   click the up-arrow beside the taskbar icons to find it.

This package includes Python, so no separate Python installation is required.

If the overlay cannot start, it displays an error dialog. You can also check
codex_usage_overlay_error.log in this folder for details.

Note: The overlay reads local Codex usage-session data from the current Windows
user account. Sign in and use Codex/ChatGPT at least once before expecting a
usage percentage.
